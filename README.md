# Login & Signup Form

A responsive and interactive Login/Signup Form built using **React.js** and **Vite**. The application allows users to switch between Login and Sign Up modes, enter their credentials, toggle password visibility, and receive success messages after form submission.

## 🚀 Features

* User Login and Registration Interface
* Dynamic Switching Between Login and Sign Up Forms
* Form Validation Using HTML5 Input Validation
* Password Show/Hide Functionality
* Loading State During Form Submission
* Success Messages After Login or Registration
* Responsive and User-Friendly Design
* Built with Modern React Hooks (`useState`)

## 🛠️ Technologies Used

* React.js
* Vite
* JavaScript (ES6+)
* HTML5
* CSS3

## ⚙️ Installation and Setup

### 1. Clone the Repository

```bash
git clone https://github.com/debanjana253048-hub/Login-Signup.git
```

### 2. Navigate to the Project Directory

```bash
cd Login-Signup
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

### 5. Open in Browser

Visit:

```text
http://localhost:5173
```

## 📸 Application Workflow

### Sign Up

* Enter Name, Email, and Password
* Click **Sign Up**
* Registration success message is displayed

### Login

* Enter Email and Password
* Click **Login**
* Login success message is displayed

### Additional Features

* Toggle password visibility using the checkbox
* Loading indicator during form submission
* Automatic form reset after successful submission
